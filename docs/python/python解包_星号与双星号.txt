================================================================================
   Python 解包：* (单星号) 与 ** (双星号)
================================================================================

缘起：register_fake 例子里写过
    a = x.new_empty(*x.shape[:-1], half)
其中 *x.shape[:-1] 就是"解包(unpacking)"。本文系统讲 * 与 ** 的区别与用法。

一句话：
    *  拆【可迭代对象】(list/tuple/Size...) → 一个个【位置参数】
    ** 拆【字典 dict】                       → 一个个【关键字参数】


================================================================================
一、起点例子：*x.shape[:-1] 在做什么
================================================================================
    a = x.new_empty(*x.shape[:-1], half)

  设 x.shape = (4, 6, 8)，half = 4：
    x.shape[:-1]        # torch.Size([4, 6])   去掉最后一维
    *x.shape[:-1]       # 把 (4, 6) 拆成两个独立参数：4, 6
    x.new_empty(*x.shape[:-1], half)
      等价于  x.new_empty(4, 6, 4)             # 4,6 是拆出来的，4 是 half

  为什么要拆：new_empty 的接口是 new_empty(d0, d1, d2, ...)——
    "一串维度大小作为位置参数"，所以要把 (4,6) 摊平成 4, 6 喂进去。

  不解包会出错：
    x.new_empty(x.shape[:-1], half)   # ❌ 把 (4,6) 当成"一个"参数传
    # new_empty 收到 ((4,6), 4)，第一个参数是元组而非整数 → 形状错误/报错


================================================================================
二、* vs ** 的核心区别
================================================================================
  ┌──────┬──────────────────────────┬──────────────┬──────────────┐
  │ 符号 │ 拆什么                    │ 拆成什么      │ 调用时写法    │
  ├──────┼──────────────────────────┼──────────────┼──────────────┤
  │  *   │ 可迭代对象(list/tuple/    │ 位置参数      │ f(*seq)      │
  │      │ Size/set/生成器...)       │              │              │
  │  **  │ 字典(dict)                │ 关键字参数    │ f(**d)       │
  └──────┴──────────────────────────┴──────────────┴──────────────┘

  例：
    def f(a, b, c): ...

    args = (1, 2, 3)
    f(*args)          # = f(1, 2, 3)         ← * 拆序列 → 按位置传

    kwargs = {"a": 1, "b": 2, "c": 3}
    f(**kwargs)       # = f(a=1, b=2, c=3)   ← ** 拆字典 → 按名字传

  混用（位置 + 关键字）：
    dims = (4, 6)
    opts = {"dtype": torch.float32, "device": "cuda"}
    x.new_empty(*dims, 4, **opts)
      # = x.new_empty(4, 6, 4, dtype=torch.float32, device="cuda")


================================================================================
三、调用时"拆开" vs 定义时"收集"（方向相反）
================================================================================
  同样的 * / ** 符号，在【函数定义】里是反向操作——把多个参数收集起来：

    def g(*args, **kwargs):
        # 调用 g(1, 2, x=3) 时：
        #   args   = (1, 2)        ← * 把多余【位置参数】收成 tuple
        #   kwargs = {"x": 3}      ← ** 把多余【关键字参数】收成 dict
        ...

  ┌──────────┬────────────────────────────┬────────────────────────────┐
  │ 出现位置 │ *                           │ **                         │
  ├──────────┼────────────────────────────┼────────────────────────────┤
  │ 调用时    │ 把序列【拆开】成位置参数    │ 把字典【拆开】成关键字参数  │
  │ (传参)    │  f(*seq)                   │  f(**d)                    │
  │ 定义时    │ 把多个位置参数【收集】成    │ 把多个关键字参数【收集】成  │
  │ (形参)    │ tuple： def g(*args)       │ dict： def g(**kwargs)     │
  └──────────┴────────────────────────────┴────────────────────────────┘

  典型转发写法（收集 + 再拆开，原样转交）：
    def wrapper(*args, **kwargs):
        return real_fn(*args, **kwargs)   # 收进来再拆出去，参数原样传递
  （vLLM 装饰器替换的 __call__(self, *args, **kwargs) 就是这个套路）


================================================================================
四、其它常见用法
================================================================================
  1) 解包赋值（星号收集中间/剩余项）
       a, *rest = [1, 2, 3, 4]      # a=1, rest=[2,3,4]
       first, *mid, last = range(5) # first=0, mid=[1,2,3], last=4

  2) 字面量里拆开拼接
       merged_list = [*l1, *l2]     # 拼接两个列表
       merged_set  = {*s1, *s2}     # 合并集合
       merged_dict = {**d1, **d2}   # 合并字典(后者覆盖前者同名键)
       new_tuple   = (*t, extra)    # 元组追加

  3) 强制关键字参数（定义里裸 * 作分隔）
       def h(a, *, b):  ...         # b 只能按关键字传：h(1, b=2)；h(1, 2) 报错

  4) 函数调用里拆多个可迭代
       print(*[1,2,3])              # = print(1, 2, 3) → 输出 1 2 3


================================================================================
五、易错点
================================================================================
  - * 只能拆【可迭代对象】；** 只能拆【dict】(且键必须是字符串、且匹配形参名)。
  - 不加 * 直接传序列，会被当成"一个参数"，而非多个。
  - f(**d) 时 d 的键要和函数形参名一致，否则报 unexpected keyword argument。
  - 顺序规则：调用时位置实参在前、* 解包、关键字、** 解包；
    定义时顺序为  def f(普通, *args, 仅关键字, **kwargs)。


================================================================================
六、一句话总结
================================================================================
  *  = 拆/收【序列 ↔ 位置参数】；   ** = 拆/收【字典 ↔ 关键字参数】。
  调用时是"拆开"(f(*seq)/f(**d))，定义时是"收集"(def g(*args, **kwargs))。
  x.new_empty(*x.shape[:-1], half) 即把 Size (4,6) 拆成 4,6 两个位置参数，
  连同 half 传给 new_empty → new_empty(4, 6, half)。
