一、LayerNorm 的数学定义
================================================================================

  对一个长度为 n 的向量 x = (x_1, ..., x_n)（这里 n = head_dim 或 hidden_dim）：

      μ   = (1/n) Σ_i x_i                 ← 算术均值
      σ²  = (1/n) Σ_i (x_i - μ)²          ← 方差（依赖 μ）
      x̂_i = (x_i - μ) / sqrt(σ² + ε)      ← 先中心化(减均值)，再除标准差
      y_i = γ_i · x̂_i + β_i               ← 可学习缩放 γ 与偏置 β

  特点：要算两个统计量（μ 和 σ²），且 σ² 依赖 μ；含"减均值"(centering)与偏置 β。

  LayerNorm 提供两类不变性（invariance）：
      - re-centering（再中心化不变性）：输入/权重整体平移，输出不变（来自减 μ）。
      - re-scaling（再缩放不变性）：输入/权重整体缩放，输出不变（来自除 σ）。


================================================================================
二、RMSNorm 的数学定义
================================================================================

  RMSNorm（Zhang & Sennrich, 2019, "Root Mean Square Layer Normalization"）
  去掉了"减均值"这一步，只保留按均方根(RMS)缩放：

      ms     = (1/n) Σ_i x_i²             ← 平方的均值(mean of squares)
      RMS(x) = sqrt(ms + ε)               ← 均方根
      x̂_i    = x_i / RMS(x)               ← 不减 μ，直接除以 RMS
      y_i    = γ_i · x̂_i                  ← 可学习缩放 γ（通常无偏置 β）

  注意：RMS 里仍然有"除以 n"（对 x² 取均值），只是它对的是 x² 而非 x。
  所以"RMSNorm 不算均值"严格说应为：不算/不减 x 的算术均值 μ（无 centering），
  但仍对 x² 取均值。

  对应本项目 kernel 实现：
      rms_rcp = rsqrtf(sumOfSquares / head_dim + eps);   // sumOfSquares=Σx²
      x[d]    = x[d] * rms_rcp * weight[d];               // weight 即 γ


================================================================================
三、逐项对比：RMSNorm 砍掉了什么、保留了什么
================================================================================

  | 步骤                | LayerNorm                  | RMSNorm              |
  |---------------------|----------------------------|----------------------|
  | 算术均值 μ=mean(x)  | 需要                       | 不需要（砍掉）       |
  | 中心化 x - μ        | 需要                       | 不做（砍掉）         |
  | 缩放分母            | sqrt(var(x)+ε)             | sqrt(mean(x²)+ε)     |
  | 可学习缩放 γ        | 有                         | 有                   |
  | 可学习偏置 β        | 通常有                     | 通常无               |
  | 统计量个数          | 2（μ 与 σ²，且 σ² 依赖 μ） | 1（mean(x²)）        |
  | re-centering 不变性 | 有                         | 无                   |
  | re-scaling 不变性   | 有                         | 有                   |

  关键观察：当 μ = 0 时，var(x) = mean(x²)，两者分母相同——RMSNorm 相当于
  "假定/不在意均值"的 LayerNorm。RMSNorm 等于把 LayerNorm 中与"均值"相关的
  全部操作（算 μ、减 μ、以及 β）拿掉，只留下按尺度归一化的核心。


================================================================================
四、为何 RMSNorm 能替换 LayerNorm
================================================================================

  RMSNorm 原论文的核心假设与发现：

      LayerNorm 之所以有效，主要来自它的 re-scaling（再缩放）不变性，
      而非 re-centering（再中心化）不变性。

  直观理由：
    1) 归一化的根本作用是"把激活的尺度(scale)控制在稳定范围"，从而稳定前向数值
       与反向梯度。承担这件事的是【除以尺度】(re-scaling)，与"减均值"关系不大。
    2) 减均值(centering)带来的 re-centering 不变性，在高维表示里贡献有限：
       - 网络可通过权重/偏置自行补偿一个整体平移；
       - 后续还有可学习 γ（及别处的 β/线性层）可吸收平移与缩放常数。
    3) 因此去掉 centering 后，模型的拟合能力几乎不受影响，却省去一次求均值与一遍
       减法（统计量从 2 个降到 1 个），计算更省、实现更简单。

  这一假设被大量经验验证（见第五节），于是现代大模型（LLaMA、GPT-NeoX、T5、
  PaLM、Gemma 等）普遍用 RMSNorm 取代 LayerNorm。


================================================================================
五、会损失精度吗？（理论 + 经验）
================================================================================

  结论先说：在有可学习参数 γ 的前提下，经验上 RMSNorm 与 LayerNorm 精度相当、
  常常持平或略优，同时显著更快——这正是它被广泛采用的原因。

  5.1 经验证据
  --------------------------------------------------------------------------------
    - RMSNorm 原论文在机器翻译、语言模型、图像分类等多任务上，效果与 LayerNorm
      相当或更好，同时报告 7%~64% 的运行时间下降（取决于模型/实现）。
    - 之后 LLaMA 等主流 LLM 全面采用 RMSNorm 并保持 SOTA 级表现，进一步证明
      用它替换 LayerNorm 在大规模训练中不损失（甚至有时改善）精度。

  5.2 为什么"砍掉中心化"几乎不掉点
  --------------------------------------------------------------------------------
    - 表示能力角度：去掉 μ 只是去掉一个"自由度"，而该自由度（整体平移）很容易被
      网络其他可学习部分（γ、相邻线性层、其它归一化处的 β）补偿。
    - 优化角度：稳定训练的关键是控制激活/梯度的尺度，由 re-scaling 提供；
      RMSNorm 完整保留了它，故训练稳定性不退化。

  5.3 诚实的边界说明
  --------------------------------------------------------------------------------
    - "不损失精度"是【经验性、在带 γ 且充分训练下成立】的结论，并非数学恒等。
      RMSNorm 与 LayerNorm 是两个不同的函数，把已用 LayerNorm 训好的权重直接换成
      RMSNorm 而不重训，可能有偏差；正确做法是用 RMSNorm 从头训练或微调。
    - 在输入均值本身携带重要信息、且无法被后续层补偿的特殊场景，理论上 centering
      可能有用；但在 Transformer 的主流用法中，这种损失未被观测到。


================================================================================
六、数值与工程角度的额外好处
================================================================================

  1) 更少计算：只需一遍 Σx²（一次归约）即可得尺度，省去求 μ 与"减均值"那一遍。
     在 GPU 上等价于少一次跨线程/跨 warp 归约与一遍逐元素减法。
  2) 实现更简单：单一统计量、无 β，kernel 更易融合（如本项目把 QK-Norm 与 RoPE
     融进一个 kernel）。
  3) 数值稳定：不做 x - μ，避免大数相减的抵消误差（catastrophic cancellation）；
     直接对平方和归约，配合 fp32 累加更稳。


================================================================================
附录 A、澄清：归一化与"正态分布"无关
================================================================================

  常见误解：LayerNorm 建立在"数据服从正态分布"的假设上，归一化后数据就服从
  N(0,1) 正态分布。——这是错的，需澄清。

  A.1 标准化 ≠ 变成正态分布
  --------------------------------------------------------------------------------

    LayerNorm 做的是标准化(z-score)：y_i = (x_i - μ)/σ。
    这是一个仿射变换(平移 + 缩放)，其关键性质：

        仿射变换只改变分布的位置(均值)和尺度(方差)，
        不改变分布的形状(偏度、峰度、是否多峰)。

    因此：
      - 输入若是偏态/双峰/均匀分布，输出仍是偏态/双峰/均匀，只是被搬到均值 0、
        缩放到方差 1；它不会把非高斯数据变成高斯。
      - LayerNorm 也【不假设】输入高斯——对任何分布都照算均值方差、照常工作。

              标准化前                标准化后(LayerNorm)
       形状:  偏态/双峰/任意    ───►   形状不变(仍偏态/双峰)
       均值:  μ                ───►   0
       方差:  σ²               ───►   1

    "以为标准化 = N(0,1)"的根源：N(0,1) 的均值 0、方差 1，于是把"均值0方差1"
    错当成"标准正态"。但无数分布都可均值0方差1而非高斯。归一化只匹配前两阶矩，
    不是对整个分布形状整形。

  A.2 RMSNorm 对分布的影响
  --------------------------------------------------------------------------------

    RMSNorm：y_i = x_i / RMS(x)，RMS = sqrt(mean(x²))，是【纯缩放】(无平移)。
    各阶量的变化：

        mean(y²) = mean(x²)/RMS² = 1            ← 二阶原点矩固定为 1(RMS(y)=1)
        mean(y)  = mean(x)/RMS                   ← 均值未清零，只被同比缩放
        var(y)   = 1 - (mean(x)/RMS)²            ← 一般 ≤ 1，仅当 mean(x)=0 时 =1

    | 方法      | 变换类型      | 改形状 | 均值              | 尺度                       |
    |-----------|---------------|--------|-------------------|----------------------------|
    | LayerNorm | 仿射(平移+缩放)| 否     | 强制 = 0(中心化)  | 方差强制 = 1               |
    | RMSNorm   | 纯缩放        | 否     | 不强制为 0        | RMS=1，var=1-(mean/RMS)²≤1 |

    结论：两者都【不改变分布形状】、都【不产生高斯】；唯一区别是 LayerNorm 额外
    做"去均值(中心化)"，RMSNorm 不做——所以 RMSNorm 相对 LayerNorm"少改"的分布
    特性就是均值(不再拉到 0)。

附录 B、矩(moment)的概念：原点矩 vs 中心矩
================================================================================

  上文反复出现"二阶原点矩/二阶中心矩(方差)"，此处补全定义。

  B.1 两套体系
  --------------------------------------------------------------------------------

    原点矩(raw moment，绕原点 0)：对 X^k 求期望
        k 阶原点矩  μ'_k = E[X^k]
          1 阶 = E[X]      ← 均值 μ
          2 阶 = E[X²]     ← 平方的均值(即"二阶原点矩")
          3 阶 = E[X³]

    中心矩(central moment，绕均值 μ)：先减均值再求 (X-μ)^k 的期望
        k 阶中心矩  μ_k = E[(X-μ)^k]
          1 阶 = E[X-μ] = 0
          2 阶 = E[(X-μ)²] ← 方差 σ²(即"二阶中心矩")
          3 阶 → 偏度(skewness)
          4 阶 → 峰度(kurtosis)

    区别只在"绕谁展开"：原点矩绕 0(直接平方)，中心矩绕均值 μ(先减均值再平方)。

  B.2 二者关系
  --------------------------------------------------------------------------------

        方差 = 二阶中心矩 = 二阶原点矩 − (一阶原点矩)²
        σ²   = E[(X-μ)²]  = E[X²]       − (E[X])²       即  Var(X) = E[X²] − μ²

      - μ = 0 时，二阶原点矩 E[X²] 就等于方差；
      - μ ≠ 0 时，二阶原点矩 = 方差 + μ²，比方差大。

  B.3 对应到 RMSNorm / LayerNorm（用样本均值代替期望：E[·] → (1/n)Σ）
  --------------------------------------------------------------------------------

      | 量                     | 定义              | 是哪个矩       |
      |------------------------|-------------------|----------------|
      | mean(x) = (1/n)Σx      | 平均值            | 一阶原点矩     |
      | mean(x²) = (1/n)Σx²    | 平方的均值        | 二阶原点矩     |
      | var(x) = (1/n)Σ(x-μ)²  | 方差              | 二阶中心矩     |

      - RMSNorm 用 RMS=sqrt(mean(x²)) 做分母 → 归一化【二阶原点矩】(令 E[y²]=1)，不减均值。
      - LayerNorm 用 sqrt(var(x)) 做分母且先减均值 → 归一化【二阶中心矩(方差)】(令 Var(y)=1)。

      所以：RMSNorm 固定二阶原点矩 = 1，LayerNorm 固定二阶中心矩(方差) = 1；
      二者只差一个 μ²，当均值为 0 时等价。


================================================================================
七、小结
================================================================================

  - LayerNorm：减均值 + 除标准差（两个统计量 μ、σ²，σ² 依赖 μ；含 γ、β）。
  - RMSNorm  ：不减均值，只除均方根 sqrt(mean(x²)+ε)（一个统计量；含 γ，常无 β）。
  - 替换可行的根据：归一化的有效性主要来自 re-scaling 不变性，RMSNorm 完整保留；
    被砍掉的 re-centering 贡献有限，且可被 γ/相邻层补偿。
  - 精度：经验上持平或略优（带 γ、从头训练/微调的前提下），并显著更快、更省、
    数值更稳——故现代 LLM 普遍用 RMSNorm 替代 LayerNorm。

  参考：Zhang & Sennrich, "Root Mean Square Layer Normalization", NeurIPS 2019。
  相关文档：本目录 qknorm&rope.txt（QK-Norm 在注意力中的特定应用）。
